1. Что такое цикл?

Цикл — это команда компьютеру повторять одно и то же действие несколько раз.

Пример из жизни: если нужно 5 раз сказать "Привет!" другу, ты можешь сделать это вручную или попросить кого-то сделать это за тебя.

for i in range(5):
    print("Привет!")

2. Два типа циклов в Python

  • for — используется, когда мы знаем, сколько раз надо повторить действие (перебрать числа, элементы списка).
  • while — используется, когда мы повторяем действие до тех пор, пока выполняется условие (например, проверка пароля).

3. Цикл for: Понятно и просто

Пример 1: Печатаем числа от 0 до 4

for i in range(5):
    print(i)

Как работает: range(5) создаёт последовательность: 0, 1, 2, 3, 4. Переменная i последовательно принимает эти значения.

Пример 2: Перебираем список

фрукты = ["яблоко", "банан", "вишня"]
for фрукт in фрукты:
    print(фрукт)

Вывод:

  • яблоко
  • банан
  • вишня

4. Цикл while: Повторяем до выполнения условия

x = 0
while x < 3:
    print(x)
    x += 1

Как работает: Проверяется условие x < 3. Если оно истинно, выполняется код внутри цикла.

5. Частые ошибки

Ошибка 1: Отсутствие отступов

for i in range(5):
print("Привет!")  # Ошибка: Нет отступа

Решение: В Python отступы обязательны.

for i in range(5):
    print("Привет!")  # Правильный код

Ошибка 2: Бесконечный цикл

x = 0
while x < 3:
    print(x)  # Ошибка: x не изменяется

Решение: Не забудьте изменять переменную внутри цикла.

x = 0
while x < 3:
    print(x)
    x += 1  # Правильный код

6. Вложенные циклы

Циклы могут быть вложены друг в друга. Пример: создадим таблицу умножения.

for i in range(1, 4):
    for j in range(1, 4):
        print(i * j, end=" ")
    print()

Вывод:

1 2 3
2 4 6
3 6 9

7. Управляющие конструкции: break и continue

Break

Прерывает выполнение цикла:

for i in range(5):
    if i == 3:
        break
    print(i)

Вывод: 0, 1, 2

Continue

Пропускает оставшуюся часть текущей итерации:

for i in range(5):
    if i == 3:
        continue
    print(i)

Вывод: 0, 1, 2, 4

8. Советы и полезные практики

  • Если цикл работает бесконечно, проверьте условие в `while` или логику в `for`.
  • Используйте вложенные циклы осторожно, чтобы не усложнять код.
  • Не забывайте о функциях break и continue для управления потоками.
